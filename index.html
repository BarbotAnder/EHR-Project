Project Abstract

The EHR Suite project is a comprehensive Electronic Health Records solution designed specifically for Integrity Mental Health in Boise, Idaho. It streamlines patient information management, appointment scheduling, and treatment optimization. This tailored software addresses gaps in existing solutions, balancing functionality with simplicity to meet the needs of practitioners and staff.

Our goal was to create a foundation for future teams to expand upon, delivering essential features such as user account registration, patient management, and secure data handling. While this is an ambitious project, it sets the stage for continued development in areas like scheduling, document management, and billing. Another big step in the devlopment of the EHR suite will be the move of the database to a cloud management system like AWS. This will improve scalability, consistency and security of critical patient details and information.

Project Description

Our project delivers a React-based web application with the following features:

  • Patient record management, including detailed notes and histories
  • Practitioner updates and version tracking for changes
  • Secure, SQLite-powered backend for fast, local data handling

Users can log in securely, manage patient records with a dynamic interface, and view changes in real-time. Designed for efficiency and simplicity, this application aims to optimize workflows in healthcare settings.
